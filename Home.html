关于 Adam Design

Adam-Design 专注于用户体验，界面设计以及一切

Adam Design 团队成立于 2016 年，由前端工程师和后端工程师构成，相对于其他的团队，我们团队更专注于改进现有的用户体验，以及打磨软件的每一项功能

现目前本团队以 React 框架为核心进行开发，利用 React 框架天生的可移植性进行跨平台开发，现目前团队正在同时开发 React 项目与微信小程序中

  • 高端的交互设计
  • 多平台同时开发
  • 引领潮流的 UI 设计
  • Git 为主的高效团队协作模式

技术栈简介

团队主要使用 React 及 React-native 进行跨平台开发，该框架为业内顶尖的 IT 公司 Facebook 所开发，维护完备，相较于以往的开发框架而言有了革命性的变化

  • 团队协作开发
  • 后期维护简单
  • 跨平台开发
  • 性能强大
  • UI成熟

通过成熟的React及React-native框架，我们可以做到同时进行多平台的开发，较之以往的开发有了长足的进步，并籍此使我们的交互设计更上一层楼。

Virtual DOM

通过Virtual DOM进行对真实网页的映射，无需花费大量时间与资源进行新功能的修改，从而极大简化后期维护成本及前期运营成本。

组件化开发

通过React的组件化特性，在团队内部分工明确，各自完成后进行对接。

Create React App

通过CRA的快速布置环境，我们得以同时进行多平台开发。
